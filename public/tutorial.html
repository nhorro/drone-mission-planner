Getting Started

Plan and export safe missions for your DJI Mini 4 Pro.

1. Prepare the map

Use the search box in the map (top left) to zoom to your takeoff area. Switch the basemap in the settings dialog if you have a MapTiler API key.

2. Add waypoints

Select Waypoints mode and click on the map to add new waypoints. Adjust altitude, speed, gimbal pitch, and actions from the edit panel.

3. Add POIs and zones

Use the POIs and Zones modes to mark points of interest and draw no-fly or capture areas that should be considered while flying.

4. Save and export

Use the toolbar buttons to save a JSON copy of your plan or export a DJI Waylines (WPML) file that can be loaded on your aircraft.
